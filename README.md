# Exception Mailer

A very simple way of sending an email in response to uncaught exceptions

## Installation

Add this line to your application's Gemfile:

    gem 'exception_mailer'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install exception_mailer
